# com-cn-guide

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态方式存放，不针对任何具体网站或域名。项目旨在提供一种简洁的页面归档方案，方便浏览和分发。

## 目录结构

```
com-cn-guide/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 公共资源文件（如图片、样式、脚本）
├── index.html      # 仓库首页（可选）
└── README.md       # 本说明文件
```

- `pages/`：每个页面独立存放，文件名即页面标识。
- `assets/`：存放页面间共享的静态资源，如 CSS、JavaScript、图片等。

## 页面归档说明

- 所有页面均为纯 HTML 文件，不依赖后端服务。
- 页面之间相互独立，无耦合，可直接在浏览器中打开查看。
- 页面内容保持原始状态，仅做必要的格式整理，不进行任何修改或优化。
- 归档遵循“一次提交，长期保存”的原则，不频繁更新页面内容。

## 维护说明

- 本仓库由维护者不定期更新，新增页面或资源。
- 欢迎通过 Issue 或 Pull Request 提出建议或提交新的归档页面。
- 提交新页面时，请确保文件格式正确，并放置于 `pages/` 目录下。
- 请勿在页面中嵌入任何形式的跟踪、广告或恶意代码。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `index.html` 或 `pages/` 下的任一 HTML 文件。
3. 也可通过 GitHub Pages 等静态托管服务部署后访问（需自行配置）。
